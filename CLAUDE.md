# Code Style Rules

- Never use JSX fragments (`<>...</>`). Use `<React.Fragment>` or a wrapper `<div>` instead.
- Never use optional chaining (`?.`). Use explicit null checks or logical AND (`&&`) instead.
- Never return explanation of things unless asked , write code. If explaination is need keep it small. when asked give normal explaination.
- Reply concisely unless asked the details
